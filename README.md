# DeepFake Detection using CNN and TensorFlow

## Overview

This project focuses on detecting DeepFake images using Deep Learning techniques. A Convolutional Neural Network (CNN) model was trained on the CelebDF V2 dataset to classify manipulated facial images as Real or Fake.

## Dataset

Dataset: CelebDF V2

https://www.kaggle.com/datasets/pranabr0y/celebdf-v2image-dataset

### Dataset Details
- Two classes:
  - Real
  - Fake
- Images extracted from videos
- Preprocessed before training

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

## Workflow

1. Load and preprocess images
2. Split data into training and validation sets
3. Train CNN model
4. Evaluate model performance
5. Classify images as Real or Fake
